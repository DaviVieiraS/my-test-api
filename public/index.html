🧪 Test API - Request Viewer

This is a serverless function that displays incoming HTTP request data.

Available Endpoints:

Simple Test 📨 POST Viewer 📝 Test Form Full Request Viewer
💡 Usage:
  • GET with query parameters: ?name=Davi&test=1
  • POST JSON data or form data
  • Both endpoints will show method, query, headers, and body

Quick Tests:

Try these URLs:

  • /api/test - Simple JSON response
  • /api/post-viewer - 📨 View POST requests with button list
  • /post-form.html - 📝 Form to send test POST requests
  • /api/handler - Full request viewer with HTML
